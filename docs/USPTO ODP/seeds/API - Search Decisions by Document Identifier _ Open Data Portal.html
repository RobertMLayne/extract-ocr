Search Decisions by Document Identifier

Use this endpoint when you want to retrieve Trial Decision data for a specific document whose document identifier you know. A document identifier is a unique ID for each document. To locate a Trial document identifier please visit search by Trial number API. You can also test the API right away in Swagger UI .

GET https://api.uspto.gov/api/v1/patent/trials/decisions/{documentidentifier}

e.g. https://api.uspto.gov/api/v1/patent/trials/decisions/171170040

Returns Trial Decision data for a document identifier. 

Note: API Key is required. Obtain an API key.

If you have any questions or errors, please visit the Support page for assistance.
